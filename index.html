First Person Horror Game

Github code Playable demo

Collaborated with a team of 3 to develop a 3d horror puzzle game built using Javascript, HTML, GLSL, WebGL, and Blender. It was made for my computer graphics final project. My job was to build the core features and come up with overall idea. I applied graphics concepts we learned in class such as matrix transformations and Phong shading to elevate the visual quality of the project. I worked on the core features including shadow and fog rendering through a 2-pass z-buffer, AABB collision detection , and basic first-person controls. I used blender and photoshop to make models and textures to achieve the horror aesthetic.

Technologies: Javascript, HTML, GLSL, WebGL, Blender
